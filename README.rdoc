== README

Basic app that takes job postings using the Import class and import entries and authors.

This app uses devise for users, anyone can create an account and apply for job postings.

Users can sort through job postings searching by keyworks and by department, as well as sort columns by asc and desc values.

Admins can manage users, postings, and import postings from the admin interface.

Deparments can review applications and move them through the process.
